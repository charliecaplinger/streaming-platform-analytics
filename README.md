# Streaming Platform Analytics

SQL and Tableau project analyzing engagement for a fictional streaming platform.

## Tools
SQL  
Tableau  

## Questions Answered

- Which shows are most popular?
- Which users watch the most content?
- Which subscription types have the highest engagement?

## Key Insights

- Identifies top-performing content
- Highlights high engagement users
- Shows subscription engagement trends

## Project Structure

data/ - sample datasets  
sql/ - analysis queries  
tableau/ - dashboard files  
dashboard_images/ - dashboard screenshots
